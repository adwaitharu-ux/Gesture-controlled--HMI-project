# Gesture-controlled--HMI-project
HMI using gesture control and haptic feedback. This system allows you to control   robotic or other systems through hand gestures and a haptic feedback is given to conform the action. 


Overview

This project is a gesture-controlled Human Machine Interface (HMI) system with haptic feedback. The system allows users to control robotic or electronic systems through hand gestures while providing tactile confirmation using vibration feedback.

Features

- Gesture-based control
- Haptic feedback confirmation
- Embedded system implementation
- Real-time interaction

Hardware Used

- ESP32 
- Gesture sensor(mpu 6050)
- Vibration motor
- flex sensor
- force sensor
- servo motors(360 servo 1,180 servo 4)

Technologies Used

- Embedded C
- Arduino IDE
- Sensor interfacing
- HMI system design

Applications

- Robotics control
- Assistive systems
- Smart interfaces
- Industrial HMI systems

Future Improvements

- Wireless communication
- AI-based gesture recognition
- Wearable integration
